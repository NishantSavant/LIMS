# Ministry Readiness Checklist (India)

Date: 2026-03-06
Project: LIMS Complete

## 1. Legal and Regulatory
- [ ] DPDP Act control mapping completed and signed off.
- [ ] Privacy notice, consent text, and data processing purpose register finalized.
- [ ] Data retention and deletion policy approved.
- [ ] Breach response SOP with escalation matrix defined.
- [ ] Terms of use and grievance officer details published.

## 2. Health and Interoperability
- [ ] EHR/EMR standards mapping documented for patient records and reports.
- [ ] Clinical data dictionaries/versioning documented.
- [ ] API schema and integration evidence prepared.
- [ ] Role-based case timeline and audit events validated.

## 3. Security Controls
- [x] Env-driven secrets and DB credentials.
- [x] Host restrictions and secure production settings baseline.
- [x] JWT auth enabled.
- [x] API throttling baseline enabled.
- [x] Structured logging baseline enabled.
- [ ] VAPT report by external assessor.
- [ ] WAF and DDoS controls at edge.
- [ ] SIEM/log retention policy and alerting runbooks.

## 4. Operations and Reliability
- [ ] Backup policy with restore drill evidence.
- [ ] Disaster Recovery RTO/RPO documented and tested.
- [ ] Monitoring dashboards (API latency, errors, DB health).
- [ ] Uptime/SLA definitions.
- [ ] Release rollback playbook.

## 5. Product Assurance
- [ ] Unit and integration test coverage target achieved.
- [ ] End-to-end scenario tests for patient/doctor/lab workflows.
- [ ] Load/performance test report.
- [ ] UAT sign-off from pilot sites.

## 6. Governance Evidence Pack (for submission)
- [ ] Architecture diagram (network, app, DB, storage).
- [ ] Access control matrix by role.
- [ ] Audit log sample exports.
- [ ] Incident and change management SOPs.
- [ ] Security hardening checklist and evidence.
